# Spring Cloud分布式商城
1.zuul网关整合spring security + JWT授权认证   
2.注册中心由eureka改为nacos
3.文件服务器使用FastDFS
4.redis缓存购物车数据以及短信验证码
5.RabbitMQ实现各服务间通讯，数据同步
6.Elastic Search实现商品搜索
